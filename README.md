# Getting Started with FIS3

FIS3 入门配置，一个简单的 React, Redux 应用

```shell
$ npm i -g fis3
```

```shell
$ git clone git@github.com:jasonliao/getting-started-with-fis3.git
$ npm install
```

```shell
$ mkdir dist
$ fis3 server start --root ./dist/
$ fis3 release
```

[http://localhost:8080]('http://localhost:8080')
